# get the course

get list class, course, industry code

Method: `POST`  
URL: `https://thongtindaotao.sgu.edu.vn/api/dkmh/w-locdsnhomto`  
Headers:  
 - `Content-Type`: `application/json`  
 - `Authorization`: `Bearer {access_token}`

Variables:  
 - `{access_token}` : Read [auth](auth.md)


Body: 
 ```json
 {
  "is_CVHT": false,
  "additional": {
    "paging": {
      "limit": 8000,
      "page": 1
    },
    "ordering": [
      {
        "name": "",
        "order_type": ""
      }
    ]
  }
}
 ```
